# CalificacionesEstudiantiles
Ejercicio de Calificaciones Estudiantiles con Windows Forms

Descripción:
Este es un programa realizado con Windows Forms que permite ingresar las calificaciones de 4 estudiantes, lo que retornará el promedio, si no es suficiente pues permite calcular las notas del completivo y si tampoco es suficiente de igual forma permite calcular el extraordinario. Al final dirá los promedios correspondientes y si el estudiante aprobó o reprobó.

Herramientas utilizadas:
- Lenguaje: C#
- Framework: .NET Framework
- IDE: Visual Studio 2022
